  3. Fruited to Return
    Unity C# Game Jam #01

    Fruited to Return is a 2D precision platformer with a Roguelite feature. A journey of a mango that is brought to an unknown world to save his avocado wife. This game was a submission to a game jam called thatgamejam #01, which gave me a lot of insight in the game and how to improve it.

  4. M2VEX (FMV School Tour)
    Unity C# Client Work

    M2VEX is a virtual campus exploration game designed to provide immersive and informative walkthrough of Mapúa Malayan Colleges Mindanao (MMCM). Set within a full-motion video (FMV) interface, the game enables users to navigate the campus environment while being guided by a virtual avatar.

  5. Cat Connect
    Unity C#

    A 32x32 2D pixel cozy game where the main goal is to maintain and grow your cafe's satisfaction level. The player must prevent the cafe’s satisfaction from dropping to zero. To keep the cafe thriving, players manage resources, upgrade equipment, and build new furniture to improve both customer experience and efficiency.

  8. Frigg's Grill n Thrill

    Welcome New Employee! You are given the chance to change your life if you work hard on Frigg's Grill N Thrill. Please follow protocols and respect the rules.... or else. This game is submitted in the Scream Jam 2024 game jam which was placed top 30.

  9. Sherlock Science
    Unity C# Educational

    A 2D game for kids. It is an educational point-and-click game where the player finds a certain item that is needed for the objective. After finding the objective, a definition of the item is shown to explain its uses.
